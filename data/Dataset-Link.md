# Dataset

Dataset is taken from [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/357/occupancy+detection) which includes 5 features:

- Temperature
- Humidity
- Light
- CO2
- HumidityRatio

and the target variable: Occupancy (0 or 1). Features are recorded from sensors and target is obtained by time stamped pictures taken every minute.

> NOTE: All data files (dataset.txt, dataset2.txt, datatraining.txt) are combined into a single dataset and then split for training and evaluation.
